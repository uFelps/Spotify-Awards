## How to run
### Backend
Requirements: Java 17
&nbsp;

```
# Clone Repository
git clone https://github.com/uFelps/Spotify-Awards

# Enter in folder backend
cd backend

# Run
./mvnw spring-boot:run
```

### Frontend
Clone Repository: NodeJS, NPM, Yarn
&nbsp;

```
# Enter in folder frontend
cd frontend

# Run
yarn start
```
